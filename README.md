# Market Price Forecasting for Farmers 🌾

This project focuses on predicting agricultural commodity prices using **time-series forecasting techniques**. The model analyzes historical market price data and forecasts future prices to help farmers and traders make informed decisions.

## 📌 Project Overview
The notebook implements a complete machine learning pipeline including:

- Data loading and preprocessing  
- Exploratory Data Analysis (EDA)  
- Time-series analysis of commodity prices  
- Forecasting using **ARIMA** and **LSTM** models  
- Model evaluation and visualization of predictions  

## 📊 Models Used
- **ARIMA** – Statistical model for time-series forecasting  
- **LSTM** – Deep learning model for capturing long-term temporal patterns  

## 🛠️ Technologies Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Statsmodels (ARIMA)  
- TensorFlow / Keras (LSTM)  
- Google Colab  


## ▶️ Run the Notebook
You can open and run the notebook using **Google Colab or Jupyter Notebook**.
